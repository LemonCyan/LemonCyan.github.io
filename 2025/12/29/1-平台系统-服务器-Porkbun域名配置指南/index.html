Porkbun 域名购买与配置指南

Porkbun 域名购买与配置详细指南

本文档旨在为新手提供详细的步骤，指导如何在 Porkbun 购买域名、配置 DNS 解析到您的服务器，并配置 SSL 加密证书。

1. 购买域名

Porkbun 是一家口碑极佳的域名注册商，价格透明，且免费提供 WHOIS 隐私保护。

步驟流程

  1. 访问官网：打开 https://porkbun.com/。
  2. 搜索域名：
    • 在首页巨大的搜索框中，输入您心仪的域名（例如 my-awesome-blog.com）。
    • 点击搜索图标。
  3. 选择域名：
    • 系统会列出搜索结果。如果域名可用，会显示绿色的 + 号和价格。
    • 点击 + 号将域名添加到购物车（Checkout）。
  4. 结账（Checkout）：
    • 点击右上角的购物车图标，选择 Checkout。
    • 注册/登录账号：
      • 如果是新用户，在弹出的窗口填写邮箱和密码进行注册（需验证邮箱）。
      • 老用户直接登录。
  5. 确认订单：
    • 检查年限（默认 1 年）。
    • Auto Renew（自动续费）：建议开启，防止域名过期被抢注。
    • WHOIS Privacy（隐私保护）：默认开启（免费），强烈建议保持开启，保护个人信息。
  6. 支付：
    • 点击 Continue to Billing。
    • Porkbun 支持 **支付宝 (Alipay)**、微信支付、信用卡、PayPal 等多种方式。
    • 对于国内用户，直接选择 Alipay 扫码支付最方便。
  7. 完成：支付成功后，您会收到邮件确认，点击右上角 Account -> Domain Management 即可看到刚购买的域名。

2. 配置域名解析 (DNS)

这一步是将您的域名（如 example.com）指向您的服务器 IP 地址（如 123.45.67.89）。

准备工作

  • 获取您 VPS 服务器的 公网 IP 地址。

配置步骤

  1. 登录 Porkbun，点击右上角 Account -> Domain Management。
  2. 找到您的域名，点击域名右侧的 Details（详情）按钮展开菜单。
  3. 找到 DNS RECORDS 一栏，点击下方的 Edit（编辑）链接。
  4. 删除默认记录（如果有）：
    • 如果看到指向 pixie.porkbun.com 或其他停车页面的记录，请点击垃圾桶图标删除，以免冲突。
  5. 添加 A 记录（指向主域名）：
    • Type: 选择 A - Address record
    • Host: 输入 @ （代表主域名，如 example.com）
    • Answer: 输入您的 服务器 IP 地址
    • TTL: 默认即可（600）
    • 点击 Add 按钮。
  6. 添加 CNAME 记录（指向 www 子域名，可选）：
    • 如果您希望用户通过 www.example.com 也能访问：
    • Type: 选择 CNAME - Canonical Name record
    • Host: 输入 www
    • Answer: 输入您的域名（如 example.com）或直接填 @ (视界面提示支持情况，Porkbun通常填域名)
    • 点击 Add。

提示：DNS 修改后需要一定时间在全球生效（通常几分钟到 1 小时，最长 48 小时）。您可以使用 ping example.com 命令检查是否已解析到您的 IP。

3. 配置 SSL 加密证书

为了让您的网站支持 HTTPS（小锁头标志），保证数据传输安全，需要配置 SSL 证书。

方案 A：使用 Porkbun 免费证书（推荐用于独立 VPS）

Porkbun 提供免费的 Let’s Encrypt 证书，您可以下载并在 Nginx/Apache 中配置。

  1. 下载证书：

    • 在 Domain Management 页面，点击域名右侧的 Details。
    • 找到 SSL 一栏。Porkbun 分为 “BUN” (Porkbun 托管) 和 “External” (外部主机)。
    • 如果是外部 VPS，点击 SSL 右侧的 Download SSL Bundle（如果还没生成，可能需要等待一两个小时自动生成，或者手动点击生成）。
    • 下载后解压，您会得到 4 个文件，主要用到：
      • private.key.pem (私钥)
      • domain.cert.pem (域名证书)
      • intermediate.cert.pem (中间证书)
      • public.key.pem (公钥，通常不用单独配)

  2. 上传到服务器：

    • 使用 SSH 连接到服务器。
    • 创建存放证书的目录：
    • 将 private.key.pem 和 domain.cert.pem 上传到该目录（可以使用 SCP 或 SFTP 工具）。

  3. **配置 Web 服务器 (以 Nginx 为例)**：

    • 编辑 Nginx 配置文件（通常在 /etc/nginx/sites-available/default 或 /etc/nginx/conf.d/your_site.conf）。
    • 添加 SSL 配置：

  4. 重启 Nginx：

    • 检查配置：nginx -t
    • 重启服务：systemctl restart nginx

方案 B：使用 Cloudflare (更简单，兼具 CDN 加速)

如果您不想手动管理证书文件，可以使用 Cloudflare。

  1. 注册并登录 Cloudflare。
  2. 点击 Add a Site，输入您的域名。
  3. 选择 Free（免费）计划。
  4. Cloudflare 会扫描您现有的 DNS 记录（确认刚才在 Porkbun 设置的 A 记录都在）。
  5. **修改 Nameservers (NS)**：
    • Cloudflare 会给出两个 Nameservers（例如 amy.ns.cloudflare.com）。
    • 回到 Porkbun Domain Management。
    • 点击 Details -> 找到 AUTHORITATIVE NAMESERVERS -> 点击 Edit。
    • 将 Porkbun 默认的 NS 删除，填入 Cloudflare 提供的两个 NS 地址。
    • 保存。
  6. 开启 SSL：
    • 在 Cloudflare 后台，点击 SSL/TLS。
    • 将模式设置为 Full (如果服务器有自签名证书) 或 Flexible (如果服务器只开了 HTTP 80 端口)。
    • 注意：为了安全，建议服务器端即使没有公网证书，也最好配置自签名证书并使用 Full 模式。如果完全不想配置服务器 SSL，使用 Flexible 模式即可实现 浏览器 -> Cloudflare (HTTPS) -> 服务器 (HTTP)。

常见问题

  • Q: 域名购买后多久能用？
    • A: 购买立即成功。DNS 解析设置后，通常 10 分钟内生效，个别地区可能需要更久。
  • Q: Porkbun SSL 证书过期了怎么办？
    • A: Porkbun 的证书是 Let’s Encrypt 的，有效期 90 天。如果您使用通过 API 的自动化工具（如 acme.sh），可以自动续期。如果是手动下载上传的（如方案 A），每 3 个月通过 Porkbun 后台重新下载并覆盖一次略显麻烦。推荐长期使用方案 B (Cloudflare) 或者在服务器上直接使用 certbot 申请证书实现自动续期。
