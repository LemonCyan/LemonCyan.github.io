Trae或VSCode下Qt环境的配置

安装必要工具

在开始配置之前，请确保安装了以下工具：

  1. Qt：Qt 是构建 GUI 应用的核心框架。下载并安装最新版本的 Qt，具体下载地址可参考 Qt 官方网站。
  2. CMake：从 Qt 6 开始，CMake 成为默认的构建工具。下载 CMake 的具体版本，可访问 CMake 官方网站。

完成安装后，确保这些工具已正确添加到系统环境变量中，以便在后续步骤中正确识别。

配置 Trae 开发环境

步骤 1：安装扩展插件

  1. 打开 Cursor 或 VS Code。

  2. 进入插件市场，安装以下必要插件：

    • C/C++ 插件：为 C/C++ 开发提供语法支持和代码补全功能。
    • Qt Configure：用于配置 Qt 开发环境。
    • Qt Tools：提供 Qt 相关工具的支持。
    • CMake：用于 CMake 构建系统的支持。
    • CMake Tools：辅助 CMake 的构建过程。

    如果某些插件在默认的插件市场中找不到，可以通过修改扩展市场地址来解决。具体操作如下：

    • 打开用户设置（快捷键 Ctrl + ,）。
    • 在搜索栏输入 application.extensionMarketUrl。
    • 修改 extensionMarketUrl 为 https://marketplace.visualstudio.com/_apis/public/gallery。

步骤 2：设置 Qt 安装路径

  1. 打开命令面板（快捷键 Ctrl + Shift + P）。
  2. 输入 >Set Qt Dir，然后选择 Qt 安装的主目录，例如 D:\Qt\5.14.2。

步骤 3：创建新 Qt 项目

  1. 再次打开命令面板，输入 >New Qt Project。
  2. 根据提示输入项目名称，选择适用的项目套件。
  3. 选择使用 CMake 或 qmake 作为构建工具，并根据项目需求选择是否包含 UI 组件。

配置 CMake

在使用 CMake 工具时，可能会遇到编译信息缺失的问题。按照以下步骤进行配置：

  1. 打开命令面板，输入 >Select CMake Kit，选择合适的 CMake 套件。
  2. 在设置中找到 CMake 路径，填写 CMake 的安装路径，例如 D:\Qt\cmake-3.31.6\bin\cmake.exe。
  3. 应用配置后，重新构建项目以确保配置生效。

下载 CMake 的具体版本可参考 CMake 官方下载页面。

配置 Qt Configure

进入 Qt configure 插件的设置界面，分别输入 MingwPath 为 D:\Qt\Qt5.14.2\Tools\mingw730_64，QtDir 为 d:\Qt\Qt5.14.2，Qt Kit Dir 为 D:\Qt\Qt5.14.2\Tools\mingw730_64，并将设置应用到所有
