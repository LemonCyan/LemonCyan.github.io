知识结构

知识结构会涉及到知识技能的深度和广度，一门语言，知识技能构成如下：

  • 数据结构 + 算法（不论在哪门语言下，其理论基础是一致的）
  • 语法 + 基础库
  • 常用框架的掌握
  • 模式和最佳实践
  • 性能调优、Debugging、Troubleshooting 等
  • 可以对语言深入吐槽，比如 python 为什么搞 GIL，.net 的 GC 线程怎么就不同呢
    学习框架源码可以让你规范自己的代码、提高代码质量。学习多个框架可以开阔自己的视野，了解方案的差异化，就很容易找到适合自己项目的方法。
