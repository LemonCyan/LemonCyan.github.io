域名类型及其注册方式

域名类型及其注册方式

域名类型

直连域名（Direct Domain）

指直接将服务器用于常规用途的情形。实际上，该域名（或子域名）直接指向服务器的 IP 地址；即执行 ping 命令时，返回的即为该服务器的 IP 地址。

CDN 域名（CDN Domain）

指服务器位于某 CDN 服务提供商之后的情形。换言之，您需先将域名（或子域名）在 CDN 提供商处注册并配置代理。本文以 Cloudflare 为例说明注册流程。若欲了解如何在已由 Cloudflare 注册的域名上启用 GCore CDN，请参阅 此链接；另可查阅 此文 以了解 Amazon CDN 域名的相关配置。

中继域名（Relay Domain）

当需借助中间服务器转发用户流量时使用此类域名。例如：您在伊朗拥有一台服务器，并已为其注册了对应域名（或子域名），此时可将该域名归入本类，以便在系统配置中自动应用相应设置。更多详情

伪造域名（Fake Domain）

用于实现“请求头伪装”（Header Trick）技术：通过伪造一个与自身无关的域名，绕过 SNI 白名单代理机制。定义此类域名后，即可查看与该技巧相关的具体配置项；但需注意，采用此方法可能导致服务器被屏蔽。更多详情

域名注册方法

在了解各类域名的基础上，下文将逐一说明其注册流程。

直连域名注册

直连域名可通过多种方式注册。

免费直连域名注册

可借助提供免费子域名的服务平台完成注册；推荐使用 afarid。该平台已预先注册若干公共主域名，用户可在其下申请所需子域名。首先须通过 此处 创建账户（需提供真实邮箱）。

注册完成后，可于 此处 查看平台已开放的主域名列表；建议选择主机数量较少、且为公开可用的域名（推荐选用 .com、.org 或 .net 后缀）。

已购直连域名注册

购买域名后，可依托第三方 DNS 服务商进行配置，例如 Cloudflare。

在 Cloudflare 上注册已购直连域名

首先通过 此链接 完成注册。

创建账户并登录后，点击 Add Site 按钮添加您的域名。

下一步选择套餐方案；免费计划已足以满足需求，按图示操作即可。

随后按图示继续操作：

最后，将系统提供的 DNS 服务器地址填入域名注册商后台设置中。

稍候片刻，待配置生效后，域名即完成在 Cloudflare 的激活。

接下来需进入 DNS 记录管理界面：

针对 IPv4 记录，请参照下图操作：

点击 “Add record” 后，填写目标子域名及对应服务器 IP 地址；务必关闭代理功能（Proxy status: DNS only），否则将无法实现直连。

如需调整 TTL 值，可将其从默认的“自动”模式改为手动设定；TTL 数值越低，用户本地缓存的 DNS 记录刷新速度越快。

直连域名注册验证

最简易的验证方式是执行 ping 命令：在终端中对目标域名发起 ping 请求，响应中应显示前述所配置的服务器 IP。通常 DNS 记录传播需一定时间；亦可借助 nslookup 或 dig 等更专业的工具进行检测。如需深入理解相关原理，可参阅 此文。

CDN 域名注册

此类域名部署于 CDN 服务提供商（即代理层）之后。以下以 Cloudflare 为例说明注册步骤。

在 Cloudflare 上注册 CDN 域名

沿用前述流程完成域名激活后，进入 DNS 设置界面添加记录：IPv4 使用 A 记录，IPv6 使用 AAAA 记录；关键区别在于需开启代理功能（Proxy status: Proxied）。

CDN 域名注册验证

使用 ping、nslookup 或 dig 等工具测试时，返回结果将为 Cloudflare 随机分配的任一 IP 地址，而不再暴露原始服务器 IP，从而增强服务器安全性。

需注意：该 IP 并非固定不变，会随时间动态轮换。在伊朗等地，Cloudflare 分配的部分 IP 可能遭遇屏蔽或干扰；对此，应结合相应技术手段规避此类限制。

CDN 域名证书配置

TLS 是一种加密互联网通信流量的协议，旨在保障用户在线安全。

具体而言，TLS 通过一套密码学算法（如 ALPN、uTLS、allowInsecure）对 Web 平台中客户端与服务器之间的通信进行加密；域名所绑定的证书亦基于 TLS 协议签发。

操作步骤如下：

  1. 进入 Cloudflare 的 SSL/TLS 设置页，将证书模式设为 Full：

  1. 启用 SSL/TLS Recommender 功能，以提升连接安全性——该功能将自动检测连接状态，并在可能升级 TLS 版本时提供建议。

  2. 转至 Network 设置页，确认 QUIC、gRPC 与 WebSockets 三项均已启用：


  1. 查看 CDN 域名流量安全状态：进入 SSL/TLS 页面，图表将展示各 TLS 版本下的流量占比——TLS 版本越高，传输安全性越强；HTTP 流量则未启用 TLS 加密。
