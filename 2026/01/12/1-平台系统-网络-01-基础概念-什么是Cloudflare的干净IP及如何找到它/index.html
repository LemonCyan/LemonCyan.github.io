什么是Cloudflare的干净IP及如何找到它

寻找纯净 Cloudflare IP 的指南

全球规模最大的 CDN 服务提供商是 Cloudflare，这一点您很可能已有所了解；由于伊朗对互联网实施了极为严格的审查过滤，Cloudflare 的服务因此频繁遭受干扰——尽管审查方无法彻底屏蔽其全部流量，却仍能造成显著的连接中断。若您亦在使用 Cloudflare 的 CDN 服务，则同样会受到此类干扰的影响。为减轻此类干扰带来的影响，您应主动搜寻“纯净 IP”（即未受干扰、可稳定访问的 IP 地址）。

以下为解决此问题的若干可行方法。

提示
在开始操作前，请务必注意：所有测试均须在未连接任何 VPN 的客户端系统上独立完成。

使用 Bashsiz 扫描器

Bashsiz 先生是一位伊朗工程师，他开发了一款名为 CFScanner 的程序，可用于在不同网络环境下测试 Cloudflare 的 IP 列表，并从中筛选出纯净 IP。

该程序已发布多个版本，适配多种平台。截至本文撰写时，可用版本包括：

  • Bash
  • Docker
  • Windows
  • Python
  • Golang
  • Android

请首先从此处下载所需版本：GitHub 项目页；随后依据目标操作系统，参照对应说明执行运行操作。

使用 Vahid Farid 扫描器

该程序已适配多平台，您可根据实际需求选择对应版本。截至本文编辑时，可用版本包括：

  • Mac
  • Windows
  • Android
  • Web

前置条件

  • 首先，根据操作系统下载并安装 Python。

  • 从此处下载源码压缩包：GitHub 项目页，解压后进入目录。

  • 安装依赖库 requests：

运行程序

  • 打开终端（例如在 Windows 中：进入解压目录 → 按住 Shift 键右击 → 选择“在此处打开命令窗口”）。

  • 输入以下命令启动程序：

    python start.py

    启动后，程序将依次引导您完成以下设置：

    • 输入所需获取的 IP 数量；

    • 设置最大允许延迟（单位：毫秒，如 500 表示 500 ms）；

    • 选择扫描 IP 段范围：多个范围以逗号分隔，排除某段则在其前加 - 符号；

    • 可排除特定干扰源：同样以逗号分隔多个范围，用 - 排除单个范围；

    • 指定探测包大小（单位：KB）：数值越大，扫描精度越高，但流量消耗亦随之上升；

    • 设定最低下载速率阈值（单位：Mbps），仅保留高于该值的 IP；

    • 设定最低上传速率阈值（单位：Mbps），逻辑同上；

    • 是否立即向 Cloudflare 注册扫描结果？若选择 y，则需继续提供以下信息：

      • 关联 Cloudflare 账户的注册邮箱；
      • 对应域名的 Zone ID（进入域名设置页面 → 查找并复制 Zone ID，参见下图）：

    • 获取全局 API Token：

      • 登录 Cloudflare 控制台主页 → 点击 Get your API token；

      • 进入 Tokens 页面 → 在 Global API Key 区域点击 View；

      • 输入账户密码后，系统将显示您的 Global API Key；

      • 将该 Token 复制粘贴至程序中；

    • 输入用于注册这些 IP 的子域名（如 cf.yourdomain.com）。

    随后扫描器将自动启动扫描流程；任务结束后，符合设定条件的 IP 将予以呈现。若此前已完成 Cloudflare 注册步骤，这些 IP 也将同步绑定至指定子域名，且代理功能保持关闭状态——即您将在 Cloudflare 中获得一个名为 cf.yourdomain.com 的子域名，其解析记录已自动指向扫描所得的纯净 IP。

使用 Safa Safarian 扫描器

该扫描器已推出多个平台版本，供您按需选用。截至本文编辑时，可用版本包括：

  • Windows
  • Mac
  • Android

脚本安装

Windows

  • 从 此处 下载 Python 安装包；

  • 安装过程中请务必勾选 “pip” 安装选项，并将 Python 添加至系统 PATH；

  • 下载原始压缩包：GitHub 主分支 ZIP，解压至本地；

  • 进入解压目录 → 按住 Shift 键右击 → 选择“在此处打开命令窗口”；

  • 执行以下命令安装依赖：

    pip install -r ./requirements.txt

    依赖安装完成后，即可进入运行阶段。

Android

  • 需先安装 Termux 应用，下载地址：GitHub Releases；

  • 根据设备架构选择对应 APK（若不确定型号，建议安装 armeabi-v7a 版本）；

  • 启动 Termux 后，在命令行中依次输入以下指令（安装过程中所有提示均回答 y）：

    pkg update -y; pkg install -y python python-pip openssl python-cryptography

  • 接着执行：

    curl -sLo main.zip https://github.com/SafaSafari/ss-cloud-scanner/archive/refs/heads/main.zip && unzip -qq main.zip && rm main.zip

  • 进入程序目录并安装依赖：

    cd ss-cloud-scanner-main
pip install -r ./requirements.txt

    依赖安装完毕后，即可运行主程序

运行脚本

  • 执行以下命令启动扫描：

    python main.py

  • 程序启动后将依次询问：

    • 所需 IP 数量；
    • 扫描模式类型；
    • 选择使用 https 或 http 协议进行探测;
    • 随后开始扫描，若干分钟后，健康 IP（如有）将列于屏幕。
    • 若选择模式 2（高级模式），则需额外输入代理的 uuid、domain、port 与 path 参数；填入后扫描随即启动——此模式将严格依据您的配置进行模拟探测。

加速模式（Speed Mode）
鉴于伊朗对 SNI 实施白名单机制，该模式采用替代型工作线程，大幅提升了扫描效率。

Vmess 模式
此模式通过构造 Vmess 数据包并经 WebSocket 发送至目标服务器，从而真实模拟代理连接行为。主流替代方案多依赖 V2Ray 或 Xray 内核实现，而本扫描器则完全基于纯 Python 实现，轻量高效。
