QCustomPlot 图表库完整指南

QCustomPlot 是面向 Qt C++ 的 2D 绑图库，用于 QtWidgets 的实时数据可视化。

架构

类结构

QCustomPlot 是主部件，内部用 Layout 系统管理可视元素。

类 功能
QCustomPlot 主部件，管理图层、坐标轴、绑定对象和交互
QCPAxisRect 坐标轴矩形，包含多个 QCPAxis
QCPAxis 坐标轴，处理刻度、标签、范围
QCPGraph 折线图/散点图，继承自 QCPAbstractPlottable
QCPLayoutGrid 网格布局，用于排列多个 AxisRect

渲染流程

默认用 QPainter 软件渲染。流程：

  1. 按 z-order 遍历 QCPLayer
  2. 每个 Layer 调用 QCPLayerable 的绑制方法
  3. lmBuffered 模式缓存静态元素

常见问题

编译错误

报错 原因 解法
undefined reference to QPrinter 缺 printsupport .pro 加 QT += printsupport
GL/gl.h: No such file 缺 OpenGL 开发库 Linux: apt install mesa-common-dev libgl1-mesa-dev
too many sections (MinGW) 对象文件太大 加 QMAKE_CXXFLAGS += -Wa,-mbig-obj
unresolved external symbol 文件没加进项目 确认 qcustomplot.cpp 在项目里

运行问题

图表不显示

内存一直涨（实时场景）

高 DPI 模糊

子线程更新崩溃

对比

QCustomPlot vs Qt Charts

方面 QCustomPlot Qt Charts
架构 独立 QWidget Graphics View
实时性能 高 (可开 OpenGL) 中等
内存 低 较高
许可证 GPLv3 / 商业 LGPLv3 / 商业
Qt 版本 5.0+ / 6.x 5.7+ / 6.x

局限

  1. 没有 QML 原生支持，需用 QQuickPaintedItem 包一层
  2. 只有 2D，没有 3D
  3. 单文件 2.8MB，编译慢
  4. GPLv3 有传染性，闭源要买商业授权
