开户

赴港旅游、购物血拼、商业谈判，甚至为孩子汇学费或生活费，或是投资港股、美股，进行境外资产配置，购买海外理财保险，这些生活中的重要事项都离不开一张香港银行卡。然而，申请香港银行账户的过程却日益复杂，为此，针对香港的 10 大银行提供了一份详细的开户攻略，帮助简化这一流程。

一、香港汇丰银行（公户&个户）

01 公司账户

开户资料：

  1. 香港公司全套资料：包括公司注册文件、商业登记证、最新报表等。
  2. 董事身份证及旅行证件：提供身份证、港澳通行证或护照的复印件，以及近三个月的个人银行流水。
  3. 合同和单据：提交最近三个月的采购合同、销售合同、发票、货运单和报关单各两份。
  4. 公司对账单：提供近半年的公司银行对账单。
  5. 大陆公司资料：包括营业执照复印件。

开户流程（过港）：

  1. 提交开户资料给银行经理进行预审。
  2. 开户经理将会电话联系董事进行 KYC 尽职调查（了解你的客户）。
  3. 审核完成后，通常会在 3-5 个工作日内安排面签。
  4. 按照约定时间到香港指定分行进行面签。
  5. 开户成功后，立即取得编码器和密码信以便后续使用。

02 个人账户

开户资料：

  1. 有效身份证、港澳通行证或护照。
  2. 进入香港时的过港小票，海关会提供。
  3. 填写银行的 KYC 表格。
  4. 提供个人地址证明，如近一个月的水电费单。

开户方案：

方案一：

  • 年龄要求：22-55 岁。
  • 存款要求：存入 50 万港币，需冻结三个月。
  • 开户方式：可以选择在国内的汇丰柜台办理。
  • 开户周期：预计为 4-6 周。
  • 开户城市：全国汇丰分行均可办理。

方案二：

  • 年龄要求：22-25 岁。
  • 存款要求：最低存款 3.5 万港币，每月至少有 3.5 万港币的交易流水。
  • 开户方式：在国内汇丰柜台办理。
  • 开户城市：北京、上海、深圳、广州、天津。

03 开户流程：

  • 内地见证：

    1. 将开户资料提交至机构进行初审。
    2. 机构将资料提交给银行审核并预约分行办理。
    3. 在指定分行办理内地汇丰账户并存入资金。
    4. 2-4 周内接到汇丰银行的电汇。
    5. 收到银行邮件并按指示操作。
    6. 接到银行的回访电话，确认开户信息。
    7. 收到实体卡后，只需激活即可使用。

  • 过港开户：

    1. 准备好开户资料。
    2. 提前预约开户时间。
    3. 亲自前往香港指定汇丰网点办理开户。
    4. 在现场开立香港汇丰银行账户。

二、香港渣打银行（公户&个户）

01 公司账户

开户资料：

  1. 香港公司资料：包括公司注册证书和最新商业登记证。
  2. 业务合同：准备两份业务合同。
  3. 提单和报关单：若有，需提供。
  4. 银行流水账单：最近三个月的流水账单。
  5. 董事和股东个人信息：包括身份证、港澳通行证或护照的复印件及三个月银行流水。

开户流程（过港）：

  1. 联系机构进行初审。
  2. 确认无误后，预约开户面签时间。
  3. 在约定的时间，亲自过港前往指定分行办理开户。
  4. 面签完成后的 2-3 周内即可完成开户。
  5. 开户后，账户激活可供正常使用。

02 个人账户

开户资料：

  1. 身份证和旅行证件：有效期需在 6 个月以上。
  2. 过港小票：海关提供的证据。
  3. 银行 KYC 表格：填写必要的客户信息。

开户流程（过港）：

  1. 提交个人开户资料。
  2. 资料审核通过后，预约面签时间。
  3. 按照预约，亲自前往香港指定分行办理。
  4. 当场完成开户，在线激活账户后即可使用。

三、中国银行-香港（个户）

01 公司账户

开户资料：

  1. 各类注册文件：包括注册证书和商业登记证等。
  2. 相关身份证件：董事和股东的身份证及港澳通行证。
  3. 业务合同：两份业务合同，若有额外支持材料可提供。
  4. 银行 KYC 表格：用于了解客户背景信息。

开户流程（过港）：

  1. 确认需求，提交相关认证材料。
  2. 初审通过后与机构签署协议。
  3. 递交资料，3 个工作日内会有面签时间。
  4. 按时携带资料前往指定分行进行面签，预计 2-3 周完成开户。

四、香港东亚银行（公户&个户）

01 公司账户

开户资料：

  1. 公司基本资料：如注册证书及商业登记证等。
  2. 业务合同：必须提供两份。
  3. 董事身份证与近半年银行流水：确保提供本人的相关财务记录。
  4. 银行 KYC 表格：用于客户身份的验证。

开户流程（过港）：

  1. 预先审核所有资料，确保完整。
  2. 审核完成后，预计 3 个工作日出面签时间。
  3. 亲自前往指定分行进行面签。
  4. 面签完成后，2-3 周内完成账户的正式开户。

02 个人账户

开户资料：

  1. 有效身份证和旅行证件。
  2. 过港小票。
  3. 银行 KYC 表格和近三个月的银行流水证明。
  4. 地址证明材料。

开户流程（过港）：

  1. 提交资料进行预审。
  2. 预审合格后预约面签。
  3. 根据预约携带资料前往香港分行进行办理。
  4. 办理完成后当场可以开立账户。

五、香港集友银行（公户&个户）

01 公司账户

开户资料：

  1. 董事个人身份证：港澳通行证或护照。
  2. 近半年的银行流水：可提供最新的财务状况。
  3. 香港公司完整资料：包括注册证书和商业登记证。
  4. 业务证明材料：如采购合同和销售合同。
  5. 国内公司资料：营业执照和业务证明材料。

开户流程（过港）：

  1. 资料进行预先审核。
  2. 确定出面签时间。
  3. 亲自过港到指定分行进行面签。
  4. 提交申请后，预计 2-3 周完成开户。

02 个人账户

开户资料：

  1. 有效身份证和通行证。
  2. 过港小票。

开户流程（过港）：

  1. 提交资料后进行审查。
  2. 资料审核通过后预约面签。
  3. 按照预约时间前往指定分行办理开户。
  4. 当场完成手续，开立账户。

六、中国银行-香港（个户）

01 个人账户

开户资料：

  1. 身份证和通行证。
  2. 过港小票。

开户流程（过港）：

  1. 准备好开户资料。
  2. 提前预约面签时间。
  3. 按照预约时间亲自前往指定分行办理。
  4. 办理完成后可直接获取账户。

七、香港招商永隆银行（个户）

01 个人账户

开户资料：

  1. 身份证及通行证。
  2. 过港小票。

开户流程（过港）：

  1. 准备开户资料。
  2. 提前预约办理时间。
  3. 按照预约时间亲自前往指定分行办理。
  4. 办理完成，当场开立账户。

八、香港花旗银行（个户）

01 个人账户

开户资料：

  1. 身份证及通行证。
  2. 过港小票。

开户流程（过港）：

  1. 准备好相关资料。
  2. 提前预约办理时间。
  3. 按照约定时间亲自前往分行办理。
  4. 当场完成开户手续。

九、交通银行-香港（个户）

01 个人账户

开户资料：

  1. 身份证、通行证。
  2. 过港小票。

开户流程（过港）：

  1. 准备开户资料并进行审核。
  2. 提前预约时间进行面签。
  3. 亲自前往香港指定分行办理。
  4. 办理完成，当场下账户。

十、中国建设银行-亚洲（公户）

01 公司账户

开户资料：

  1. 香港公司基本资料，包括注册证书和商业登记证。
  2. 两套业务合同。
  3. 董事及股东身份证和通行证。
  4. 银行 KYC 表格。
  5. 国内公司营业执照及近半年流水。

开户流程（过港）：

  1. 提交资料进行预审。
  2. 确认资料完整后预约面签时间。
  3. 按时亲自过港至指定分行进行面签。
  4. 面签后 2-3 周内完成开户。

在香港，金融监管制度严格，银行在开户时常常会向申请者推荐理财产品或基金，这是因为开户经理有销售这些产品的任务。成功开户不仅是一项成就，更是一种生活的便利。希望每一步都能顺利进行，为未来的投资与生活提供更多可能。
