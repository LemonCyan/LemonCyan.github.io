7. 安装并登录 WordPress 网站

在您的本地浏览器中访问 ECS 实例的公网 IP，这将引导您进入 WordPress 的安装页面。按照说明填写网站的基本信息，然后单击“安装 WordPress”按钮。

填写信息参数说明：

  • 站点标题：您 WordPress 网站的名称，例如：demowp。
  • 用户名：用于登录 WordPress 的用户名，建议选择安全性高的名称，例如：testwp。
  • 密码：登录 WordPress 所需的密码，强烈建议使用安全性高的密码，例如：Wp.123456。
  • 您的电子邮件：用于接收重要通知的电子邮件，例如：1234567890@aliyun.com。

完成信息填写后，单击“登录”按钮。输入您刚刚设置的用户名和密码，成功后您将进入 WordPress 的管理后台。
