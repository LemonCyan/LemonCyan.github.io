4. Fluidd配置和使用

Posted on 2024-03-18 In 3D打印机

Fluidd

Fluidd是用于3D打印机Klipper固件的轻量化的快速的用户接口

KIAUH

需要安装git。

  1. 安装 git 后，克隆 KIAUH 项目git clone https://github.com/dw-0/kiauh.git
  2. 启动KIAUH，./kiauh/kiauh.sh
  3. KIAUH 的主菜单中。您将看到多个操作可供选择，具体取决于您想要执行的操作。要选择操作，只需在“执行操作”提示中输入相应的数字，然后按 ENTER 确认即可。

手动安装

Fluidd 附带一个build脚本，可在GitHub 版本( fluidd.zip) 中找到。

需要安装NodeJS (v16.x) 和 Git

  1. 克隆Fluidd 源代码：git clone https://github.com/fluidd-core/fluidd.git
  2. 导航到 Fluidd 源代码目录：cd fluidd
  3. 安装依赖项：npm ci
  4. 构建并捆绑 Fluidd：npm run build

构建的文件将写入该dist目录。您可以使用您首选的 HTTP 服务器来提供这些服务，例如NGINX。

要出于开发目的构建 Fluidd，请运行npm run serve而不是npm run build启用热重载。
